# leetcode-reviewer

A full-stack spaced-repetition app for reviewing solved LeetCode problems, modeled after Anki-style review scheduling.

## Stack
- Frontend: Next.js, React, TypeScript, Tailwind
- Backend: FastAPI, Python, raw SQL via psycopg2
- Database: PostgreSQL
- Auth: JWT (planned)

## Status
V0 in progress: basic CRUD loop (Add/View/Edit/Delete Problem) end-to-end through Frontend Form → FastAPI → PostgreSQL → JSON → UI.
